Spaceify AI Property Management

Project information

  • Category: Full-Stack / Machine Learning
  • Project URL: https://github.com/tommasaurus/AiPropertyDevelopment
  • Website: https://www.spaceify.ai/

Project Overview:
Spaceify is an AI-powered property management platform designed for small property owners to simplify and automate property management. The platform streamlines the onboarding process by allowing users to upload a lease, which is automatically parsed to create tenants, add properties, track rent, and store documents. Key dates, such as lease end dates, are added directly to the integrated calendar. Spaceify also offers financial tracking and an AI chatbot to assist with any queries about the platform, making property management effortless and efficient.

Key Features:

  • AI-Powered Lease Parsing: Automatically extracts key information from uploaded leases to create tenants, properties, rent schedules, and important dates.
  • Integrated Calendar: Automatically updates with lease end dates, rent due dates, and other key events to help users stay organized.
  • Document Management: Stores all uploaded documents securely, ensuring easy access and organization.
  • Financial Tracking: Tracks income and expenses by property and provides insights into net income, growth, and property-level performance.
  • AI Chatbot Assistance: Answers user queries about the platform and provides helpful insights to simplify property management tasks.
  • Scalable Dashboard: Displays an overview of properties, tenants, finances, and other critical information in an intuitive layout.
  • Secure Payment Integration: Stripe powers secure payment handling for subscription-based access.
  • Multi-Platform Support: Responsive design ensures seamless performance across all devices.

Technologies Used:

  • Frontend: React.js for a fast, dynamic, and user-friendly interface.
  • Backend: Python (FastAPI) and Node.js for robust server-side operations and API handling.
  • AI Integration: OpenAI for the chatbot and AI-powered automation features.
  • Database and Storage: AWS RDS for database management and S3 for secure document storage.
  • UI Design: Tailwind CSS for a clean and modern user interface.
  • Payment Integration: Stripe for secure subscription and payment handling.
  • Hosting: AWS EC2 for backend deployment and Vercel for frontend hosting.
  • Networking: Cloudflare for secure and fast networking.

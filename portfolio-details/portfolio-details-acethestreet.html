AceTheStreet Investment Banking Prep

Project information

  • Category: Full-Stack
  • Project URL: https://github.com/tommasaurus/AceTheStreet

Project Overview:
AceTheStreet is an investment banking preparation platform designed to help aspiring professionals succeed in interviews. The platform offers the most up-to-date questions from recent investment banking recruiting cycles, ensuring users have access to the most relevant material. AceTheStreet takes a gamified approach to learning, providing engaging tools such as flashcards, test and match game modes, and a robust question bank, including firm-specific questions. By combining cutting-edge technology with user-friendly features, AceTheStreet simplifies the interview preparation process while maintaining a high standard of quality.

Key Features:

  • Comprehensive Question Bank: Access a diverse set of questions, including firm-specific and category-level problems, tailored for investment banking interviews.
  • Gamified Study Modes: Engage in interactive learning with study modes like flashcards, quizzes, and matching games that make preparation both fun and effective.
  • Category-Level Filtering: Focus on specific topics or categories to target your areas of improvement and strengthen key concepts.
  • Subscription-Based Access: A seamless subscription model allowing users to unlock all features and content.
  • Modern Design and Functionality: Clean, intuitive design optimized for an enjoyable user experience across devices.
  • Recurring Revenue Model: Supporting the platform's growth with over 500+ paid subscribers.
  • Secure Payment Integration: Stripe powers safe and secure subscription transactions.
  • Responsive Performance: Built with Next.js and deployed on Vercel for fast, scalable, and reliable performance.

Technologies Used:

  • Frontend: React.js and Next.js for a fast, interactive, and responsive interface.
  • Backend: Node.js for a robust server-side application.
  • Database: Supabase for real-time database management and authentication.
  • UI Design: Tailwind CSS for a modern, responsive, and clean design.
  • Payment Integration: Stripe for secure subscription and payment handling.
  • Hosting: Vercel for fast, reliable, and scalable deployment.
  • Programming Language: TypeScript for type-safe and maintainable code.
